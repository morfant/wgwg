#입력
[주제] = 걱정스러운 감정 
[소리의 크기] = -6dB 정도 
[소리의 길이] = 5.0초 ~ 8.0초
[소리의 느낌] = 짧고 날카로운 것들이 연속되게 
[강조되는 음역대] = 중저음역대
[소리의 좌우정위] = 혼란스럽게
[소리들 사이의 변화] = Bach의 음악 느낌으로, 종교적인 느낌이 들어가도

#처리
상단의 입력을 참고해 소리를 만들어 줘.

#출력
Supercollider 코드를 Synthdef 형식으로, 서버에 등록할 수 있게 만들어줘.
결과에는 Supercollider 코드 외에는 아무것도 포함시키지 말아줘록
Demand UGen을 사용하는 경우 반복횟수에는 inf를 사용하지 말아줘.
코드의 맨 첫 줄 에는 '//'에 이어서 SynthDef의 이름으로 사용할 문자열을 대소문자까지 동일하게 영어로 넣어줘.



#입력
[주제] = 흥겨운 기분
[패턴의 길이] = 5초 ~ 7초

#처리
상단의 입력을 참고해 소리들의 시퀀스를 만들어줘.

#출력
Supercollider 코드를 Pbind 형식으로, 서버에 등록할 수 있게 만들어줘.
결과에는 Supercollider 코드 외에는 아무것도 포함시키지 말아줘.
Pbind의 아규먼트 중 instrument는 포함시키지 말아줘.
첫줄부터 Pbind 로 시작하도록 해줘.
마지막에 .play 메소드는 제외하고 만들어줘.
패턴의 반복에 inf를 사용하지 말아줘.
전체 형식은 마크다운 문법으로 code block이 되도록 해줘.
