입력
[주제] = 더러운 기분
[패턴의 길이] = 15초 ~ 27초

#처리
상단의 입력을 참고해 소리들의 시퀀스를 만들어줘.

#출력
Supercollider 코드를 Pbind 형식으로, 서버에 등록할 수 있게 만들어줘.
결과에는 Supercollider 코드 외에는 아무것도 포함시키지 말아줘.
Pdef안에 들어가는 Pbind의 아규먼트 중 instrument는 포함시키지 말아줘.
마지막에 .play 메소드는 제외하고 만들어줘.
패턴의 반복에 inf를 사용하지 말아줘.
전체 형식은 마크다운 문법으로 code block이 되도록 해줘.
코드의 맨 첫 줄 에는 '// Pbind'라고 적어줘.
코드의 두 번째 줄 에는 '// '에 이어서 패턴의 이름으로 사용할 문자열을 대소문자까지 동일하게 영어로 넣어줘.
